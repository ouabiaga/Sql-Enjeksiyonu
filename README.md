# SQL Enjeksiyonu (SQL Injection)

Bu proje, SQL enjeksiyonu (SQL Injection) güvenlik açığını anlamak, tanımlamak ve önlemek için eğitim amaçlı bir kaynaktır.

## 🎯 Amaç

- SQL enjeksiyonu nedir ve nasıl oluşur
- Güvenlik açığının potansiyel tehlikeleri
- Korunma yöntemleri ve best practices
- Pratik örnekler ve senaryolar
